    保費走勢

    依目前組合，25歲開始才會超過2萬，不過我們可能會從第1歲開始減掉一項$2400；且每三年可能一調

    審約

    全球人壽

    全球人壽失扶好照終身健康保險(G版)

    • LDG | 主約, 30年期, 1.5萬 | 3,090元
      • 故無解約金
      • 成立後31天始成效
      • 意外傷害不賠
      • 滿期日 = 110歲
      • 30歲繳滿後賠更多
      • 小殘(7-11級) 一次性給$45000
      • 重殘(1-6級) 一次性給$45000 + 重大失能一次性$360000 + 每月給付 $15000~18750 直到超過理賠總額 ( 失能第41年 )
      • 全球官網找不到費率表，是去Finfo找到的

    全球人壽傷害保險附約 → 意外失能

    • XAR ｜附約, 一年期, 200萬 320元
    • #保證續保
    • 繳費一年保障一年，每年保費可能會變動，你可以續保到年齡上限，不用找業務重新簽約。
    • 費率會因職業類別有差，內勤和家管當然是風險最少、費率最低的。但人的職業是可能變動的，難道是每年續約時依當年職業來調整？ ( http://mpost3.post.gov.tw/webpost/doc/iClassify.html )
    • XAR保的是身故&失能，醫療則是它的附約 – XMR
    • 身故的部分，因為是15歲後才有賠，所以15歲前後保費有個大落差
    • 意外發生後180天內出事，才算數 → 所以意外發生日如何判定???

    全球人壽傷害保險醫療保險金附加條款 → 意外實支實付 ( 所以疾病不算？)

    • XMR ｜ 附加/批註, 一年期, 3萬 378元
    • #副本理賠
    • 這條就是實支實付的部分，但DM上看不到有說可副本理賠 ?????
    • 限0-69歲
    • 以投保3萬元為例，因意外導致的醫療費如果是2萬元，則理賠2萬元；若是4萬元，則最多理賠3萬元
    • 累積最高保額為傷害險主、附約「累積保額」之1/10，且累積本公司及同業之投保限額為20萬元。 → ?????
    • 同一次傷害的給付總額不得超過保險單所記載的「每次實支實付傷害醫療保險金限額」 → ?????

    • #概括式 #副本理賠 #門診手術 #門診手術雜費 #保證續保

    全球人壽失扶65定期健康保險附約

    • XDK ｜ 附約, 30年期, 2.5萬 525元
    • #不還本 #1-6級失能豁免 #失能扶助金不打折
